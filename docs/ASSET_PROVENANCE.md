# Asset Provenance Log

All external assets used in this mod are tracked here for copyright clearance purposes.

## Format

Each asset entry records:
- **Asset**: filename and path within the mod
- **Source URL**: where the asset was obtained
- **Author/Creator**: who made the original
- **Licence**: licence type at time of access (e.g., CC BY 4.0, CC0, etc.)
- **Date Accessed**: when we retrieved it
- **Notes**: any modifications made, usage restrictions, etc.
